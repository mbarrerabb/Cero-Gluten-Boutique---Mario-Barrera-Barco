# Cero Gluten Boutique

**Cero Gluten Boutique** es una tienda online de cestas gourmet sin gluten, posicionada como una opción de regalo premium y saludable, dirigida tanto a personas celíacas como a cualquiera que quiera hacer un regalo diferente y consciente.

## 🚀 Características Principales

- **Diseño Premium**: Interfaz moderna y elegante con enfoque en la experiencia visual.
- **Navegación Intuitiva**: Menú claro y accesible con categorías bien definidas.
- **Secciones Destacadas**:
    - **Inicio**: Presentación de la marca y propuesta de valor.
    - **Cestas**: Catálogo de productos con detalles y precios.
    - **Suscripción**: Opciones de suscripción mensual/trimestral.
    - **Blog**: Artículos y consejos sobre vida sin gluten.
    - **Nosotros**: Historia y valores de la empresa.
    - **Contacto**: Formulario y datos de contacto.
- **Responsive Design**: Adaptado para dispositivos móviles y de escritorio.
- **Iconografía Moderna**: Uso de Font Awesome para elementos visuales atractivos.

## 🛠️ Tecnologías Utilizadas

- **HTML5**: Estructura semántica del sitio.
- **CSS3**: Estilos personalizados, animaciones y diseño responsivo.
- **Font Awesome**: Iconos vectoriales para una mejor experiencia de usuario.
